Análise de Dados HPLC

Uma ferramenta otimizada para processamento de dados de cromatografia.

Guia de Início Rápido

Bem-vindo à ferramenta de Análise de Dados HPLC! Siga os passos abaixo para processar e visualizar seus dados de forma eficiente.

Passo 1: Configurar Parâmetros das Moléculas

Navegue até a aba "Parâmetros". Aqui, você deve cadastrar cada molécula de interesse que deseja analisar em suas amostras.

  • Nome da Molécula: Um identificador único (ex: Glicose, Frutose).
  • Tempo de Referência: O tempo de retenção esperado para a molécula (em minutos). A ferramenta buscará o pico mais próximo a este tempo.
  • Fator de Conversão: O valor usado para converter a área do pico em concentração.
  • Fator de Diluição: Se a sua amostra foi diluída, insira o fator aqui (ex: 10 para uma diluição 1:10).
  • Fator de Correção: Um valor adicional para ajustar o cálculo final da concentração, se necessário.

Clique em "Adicionar Molécula" para salvar. Você pode adicionar quantas moléculas precisar.

Passo 2: Processar Amostras

Vá para a aba "Amostras". Nesta seção, você irá inserir os dados brutos do seu equipamento de HPLC.

  • Nome da Amostra: Defina um nome base para seu conjunto de amostras (ex: Cultura_24h). A ferramenta adicionará automaticamente um sufixo para cada replicata (ex: Cultura_24h_1, Cultura_24h_2).
  • Número de Replicatas: Informe quantas replicatas existem nos dados que você vai colar.
  • Dados HPLC: Cole os dados diretamente do seu software. O formato esperado é uma lista de tempos de retenção e áreas, separados por um cabeçalho para cada replicata.
1  Ret. Time  Area\n   7.438     559638\n   8.080     95873\n2  Ret. Time  Area\n   7.441     559900\n   ...

Após preencher, clique em "Processar Amostra". Os resultados serão calculados e você será levado para a aba de resultados.

Passo 3: Analisar os Resultados

Na aba "Resultados", você encontrará uma tabela consolidada com a média e o desvio padrão das concentrações para cada molécula em cada conjunto de amostras.

  • Tabela de Resultados: Visualize os dados calculados de forma clara.
  • Copiar para CSV: Use este botão para copiar os dados da tabela e colar facilmente em softwares como Excel ou Google Sheets.
  • Excluir: Remova um conjunto de resultados e todas as suas amostras associadas.
Passo 4: Criar Gráficos Comparativos

A aba "Gráfico" é onde você pode visualizar e comparar seus dados.

  • Selecione a Métrica (Eixo Y): Escolha qual dado você quer plotar (ex: Média de Glicose).
  • Monte os Conjuntos: Esta é a parte mais poderosa. Você pode criar múltiplos "conjuntos" de dados para comparação.
    • Selecione as amostras para o Conjunto 1 na ordem desejada. Elas formarão a primeira série no gráfico.
    • Clique em "Adicionar Conjunto de Amostras" para criar o Conjunto 2.
    • Selecione as amostras para o Conjunto 2. Elas serão plotadas nas mesmas posições do eixo X que as amostras do Conjunto 1, permitindo uma comparação direta.
    • O rótulo do eixo X mostrará qual amostra de cada conjunto está sendo comparada (ex: AmostraA / AmostraB).
  • Tipo de Gráfico: Alterne entre Barras e Linhas para a melhor visualização.
